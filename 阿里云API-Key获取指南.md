## 获取 API Key 步骤

### 第一步：注册阿里云账号

1. 访问阿里云官网：https://www.aliyun.com
2. 点击右上角「免费注册」
3. 按提示完成注册（需要手机号验证）

**注意**：如果您已经有阿里云账号，可以直接登录。

---

### 第二步：开通百炼服务

1. 登录阿里云后，访问百炼控制台：https://bailian.console.aliyun.com/
2. 首次访问会提示「开通服务」
3. 点击「立即开通」
4. 阅读并同意服务协议
5. 点击「确认开通」

**重要**：开通服务是免费的，不会产生费用。

---

### 第三步：获取 API Key

1. 在百炼控制台，点击左侧菜单「API-KEY管理」
2. 点击「创建 API Key」按钮
3. 输入 API Key 名称（例如：`轻食记APP`）
4. 点击「确认」
5. **立即复制 API Key**（只显示一次，请务必保存！）

**API Key 格式**：以 `sk-` 开头的一串字符，例如：
```
sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

---

### 第四步：在轻食记中配置 API Key

1. 打开轻食记 APP
2. 进入「设置」→「AI设置」
3. 选择「AI 提供商」为「阿里云 - 通义千问 Qwen3.6 Flash」
4. 在「API 密钥」输入框中粘贴刚才复制的 API Key
5. 点击「保存」

## 常见问题

### Q1：API Key 忘记了怎么办？

**A**：可以在百炼控制台 → API-KEY管理 中查看或重新创建。

### Q2：如何查看剩余额度？

**A**：访问百炼控制台 → 「用量管理」中查看。

### Q3：API Key 泄露了怎么办？

**A**：立即在百炼控制台 → API-KEY管理 中删除旧 Key，并创建新 Key。

### Q4：识别不准确怎么办？

**A**：尝试以下方法：
- 拍照时尽量清晰，光线充足
- 在「添加详情」中补充说明，例如：「大份」、「用橄榄油烹饪」
- 可以放一个叉子或勺子在食物旁边，帮助 AI 判断份量

### Q5：支持哪些食物？

**A**：通义千问对**中国食物**识别效果最好，包括：
- 中式菜肴（宫保鸡丁、麻婆豆腐等）
- 地方特色（火锅、烤鸭、小笼包等）
- 街边小吃（煎饼果子、肉夹馍等）
- 西餐、日料、韩料等
