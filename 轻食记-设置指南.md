## 🔑 第一步：获取 Gemini API Key

### 为什么选择 Gemini？
- ✅ **完全免费**：每天 1500 次请求，足够日常使用
- ✅ **识别准确**：Google 最新 AI 模型，支持中英文
- ✅ **无需绑卡**：不像 OpenAI 需要国际信用卡

### 获取步骤：

1. **访问 Google AI Studio**
   ```
   https://aistudio.google.com/app/apikey
   ```

2. **登录 Google 账号**
   - 使用你的 Gmail 账号登录
   - 如果没有 Gmail，需要先注册一个

3. **创建 API Key**
   - 点击 "Create API Key"
   - 选择 "Create API Key in new project"
   - 复制生成的 API Key（以 `AIza` 开头）

4. **保存 API Key**
   - 将 API Key 保存到安全的地方
   - 稍后在 APP 中需要输入

### Gemini 免费额度说明：

| 模型 | 每分钟请求 | 每天请求 | 每分钟 Token |
|------|-----------|---------|-------------|
| gemini-2.0-flash-lite | 15 | 1500 | 1,000,000 |
| gemini-2.0-flash | 10 | 1500 | 1,000,000 |

**估算**：每天拍照识别 50 次食物，一个月也用不完免费额度！

## 📦 第二步：运行项目

### 前置条件（已满足）
- ✅ Node.js v22.16.0
- ✅ Git 2.53.0
- ✅ 项目已克隆到 `D:\kalory-app`
- ✅ 依赖已安装（959 个包）

### 运行步骤：

#### 方式一：在手机上运行（推荐）

1. **安装 Expo Go APP**
   - iOS: App Store 搜索 "Expo Go"
   - Android: Play 商店搜索 "Expo Go"

2. **启动开发服务器**
   ```powershell
   cd D:\kalory-app
   npx expo start
   ```

3. **扫描二维码**
   - 终端会显示二维码
   - iOS 用户：用相机 App 扫描
   - Android 用户：在 Expo Go 中扫描

4. **输入 Gemini API Key**
   - 进入 APP 后，点击 "设置" → "AI 设置"
   - 选择 "Google Gemini"
   - 输入你之前获取的 API Key

#### 方式二：在电脑浏览器预览

```powershell
cd D:\kalory-app
npx expo start --web
```

按 `w` 键在浏览器中打开。

## ❓ 常见问题

### Q1: API Key 输入后报错？
- 检查 API Key 是否正确复制（以 `AIza` 开头）
- 确保网络能访问 Google 服务
- 尝试重新生成 API Key

### Q2: 拍照识别不准确？
- 确保光线充足，食物清晰可见
- 尝试添加文字描述（如："一碗米饭，约200克"）
- 可以手动编辑识别结果

### Q3: 如何备份数据？
- 数据存储在本地，卸载 APP 会丢失
- 建议定期导出数据（设置中有导出功能）
